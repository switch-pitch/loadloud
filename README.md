# loadloud (Next.js)

Проект переведен на Next.js App Router и готов для деплоя на Vercel.

## Локальный запуск

```bash
npm install
npm run dev
```

Открыть: `http://localhost:3000`

## Прод

```bash
npm run build
npm run start
```

## Деплой на Vercel

```bash
npx vercel
```

## Где менять видео и подписи

Файл: `app/page.jsx`  
Массив: `VIDEO_ITEMS`
